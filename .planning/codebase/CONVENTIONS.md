# Coding Conventions

## General Principles
- **AI-First**: Every core feature should have an intelligence layer.
- **Premium Aesthetics**: High-fidelity UI with `backdrop-blur-md`, subtle gradients, and smooth animations.
- **Type Safety**: TypeScript is mandatory for the frontend.

## Frontend (React)
- **Component Pattern**: Functional components with Tailwind CSS.
- **Animations**: Standardize on `framer-motion` for transitions.
- **UI Consistency**: Use `ui/` folder atomic components.
- **Icons**: Use `hugeicons-react` for all iconography.

## Backend (Node.js)
- **Controller-Service Pattern**: Controllers handle HTTP, Services handle logic/AI.
- **Error Handling**: Use centralized error middleware and standard response formats.
- **Naming**: camelCase for variables/functions, PascalCase for classes/models.

## Naming Conventions
- Mission Board (Planner)
- Resource Center (Marketplace)
- Tactical Hub (Dashboard)
- Strategic Map (Map Intelligence)
