# Strategic Technology Stack

## Frontend (Client Layer)
- **Framework**: React 18+ (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Hugeicons-React
- **Routing**: React Router DOM v6+

## Backend (Service Layer)
- **Runtime**: Node.js v20+
- **Framework**: Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Authentication**: JWT (JSON Web Tokens)
- **AI Engine**: NVIDIA NIM (via official SDK/API)

## AI Systems
- **Primary Intelligence**: NVIDIA NIM (LLM)
- **Backend Services**:
  - `ai.service.js`: Central logic for mission analysis, career paths, and tactical intelligence.
  - `ai.controller.js`: API endpoints for AI-driven features.

## Development & DevOps
- **Package Manager**: npm
- **Environment Management**: .env
- **Scripting**: PowerShell (launch/start scripts), JavaScript/MJS (utility scripts)
